Comprehensions

Comprehensions allow us to construct one sequence from another.

They use the `for` keyword to spin through the original sequence

Simple Example

  • First we convert a string to a list with `split()`
  • Then we use the `for` keyword to spin through the list
sentence = 'look before you spoil the broth'
words = sentence.split()
for word in words:
print(word)

Single Line - more complex

The example below does all the work on 1 line

  • Create a list of vowels to use in the search
  • Use the `for` keyword to spin through the letters in the input string
  • use lower() to convert each letter to lower case for the case-sensitive search
  • The `if` clause filters out the letters we return-in this case only returning non-vowels letters
  • We don't want a list of letters. we want a boolean True/False
    • `True` if there are only vowels in the input string
    • `False` if any of the letters in the input string are not vowels
  • We make this decision by testing the length of the result string.
    If it's 0, the comprehension didn't return any letters, so we must have only vowels!
VOWELS = 'aeiou'


def contains_only_vowels(input_str):
"""Receives input string and checks if all chars are
VOWELS. Match is case insensitive."""
return len([letter for letter in input_str.lower() if letter not in VOWELS]) == 0
